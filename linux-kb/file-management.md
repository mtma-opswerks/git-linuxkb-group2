# 📂 File Management Commands

A quick reference guide for the most common and useful Linux file management commands.

**Command:** `mkdir`  
**Description:** Create a new directory (folder) in the current location.  
**Example:** `mkdir projects`  
**Note:** Creates a directory named **projects**.

---

**Command:** `touch`  
**Description:** Create a new empty file, or update the last-modified timestamp of an existing file.  
**Example:** `touch notes.txt`  
**Note:** Creates an empty file called **notes.txt** if it doesn’t exist, or updates its timestamp.

---

**Command:** `ls -a`  
**Description:** List all files and directories, including hidden ones (those starting with a dot).  
**Example:** `ls -a`  
**Note:** Shows all files, including **.bashrc**, **.gitignore**, and other hidden files.
